Autenticação

Todos os requests precisam de um token. Gera um na secção Tokens.

HTTP
# Header (recomendado) Authorization: Bearer apl_xxxxx # Ou query param GET /api/produtos?token=apl_xxxxx

Endpoints

GET/api/{coleção}Listar todos
GET/api/{coleção}/{id}Obter um
POST/api/{coleção}Criar
PUT/api/{coleção}/{id}Substituir
PATCH/api/{coleção}/{id}Atualizar campos
DEL/api/{coleção}/{id}Eliminar

Exemplos de código

JavaScript / Fetch
const BASE = '/api'; const TOKEN = 'apl_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'; const H = { 'Authorization': `Bearer ${TOKEN}`, 'Content-Type': 'application/json' }; // Listar todos const { data } = await (await fetch(`${BASE}/produtos`, { headers: H })).json(); // Criar registo const novo = await (await fetch(`${BASE}/produtos`, { method: 'POST', headers: H, body: JSON.stringify({ nome: 'Teclado', preco: 89.99 }) })).json(); // → { id: 'rec_xxx', nome: 'Teclado', ... } // Atualizar await fetch(`${BASE}/produtos/${novo.id}`, { method: 'PUT', headers: H, body: JSON.stringify({ nome: 'Teclado Pro', preco: 99.99 }) }); // Eliminar await fetch(`${BASE}/produtos/${novo.id}`, { method: 'DELETE', headers: H });
Python
import requests BASE = '/api' H = {'Authorization': 'Bearer apl_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'} # Listar data = requests.get(f'{BASE}/produtos', headers=H).json()['data'] # Criar novo = requests.post(f'{BASE}/produtos', headers=H, json={'nome': 'Teclado', 'preco': 89.99}).json() # Atualizar requests.put(f'{BASE}/produtos/{novo["id"]}', headers=H, json={'preco': 79.99}) # Eliminar requests.delete(f'{BASE}/produtos/{novo["id"]}', headers=H)
cURL
# Listar curl /api/produtos \ -H "Authorization: Bearer apl_xxx" # Criar curl -X POST /api/produtos \ -H "Authorization: Bearer apl_xxx" \ -H "Content-Type: application/json" \ -d '{"nome":"Teclado","preco":89.99}' # Atualizar (PUT = substituição completa) curl -X PUT /api/produtos/rec_xxx \ -H "Authorization: Bearer apl_xxx" \ -d '{"nome":"Teclado Pro","preco":99.99}' # Eliminar curl -X DELETE /api/produtos/rec_xxx \ -H "Authorization: Bearer apl_xxx"
PHP
<?php $base = '/api'; $token = 'apl_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'; $h = ["Authorization: Bearer $token", 'Content-Type: application/json']; // Listar $ch = curl_init("$base/produtos"); curl_setopt_array($ch, [CURLOPT_HTTPHEADER => [$h[0]], CURLOPT_RETURNTRANSFER => true]); $data = json_decode(curl_exec($ch), true)['data']; // Criar $ch = curl_init("$base/produtos"); curl_setopt_array($ch, [CURLOPT_POST => true, CURLOPT_HTTPHEADER => $h, CURLOPT_RETURNTRANSFER => true, CURLOPT_POSTFIELDS => json_encode(['nome' => 'Teclado', 'preco' => 89.99])]); $novo = json_decode(curl_exec($ch), true);

{ "data": [ { "id": "rec_abc123", "nome": "Teclado", "preco": 89.99 } ], "count": 1, "collection": "produtos" }

404 — Registo não encontrado
400 — JSON inválido no body
405 — Método HTTP não suportado
